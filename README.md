# Discord Reminder Bot

## 특정 시간마다, 특정 채널에, 특정 메세지를 전달하는 디스코드 봇.

### 특징

- Command Cog 사용. 모듈 분리 적용. (간단하지만 새로이 적용한 개념)
- 데이터는 Notion에서 가져오도록 함.
- Task를 적용하여 일정 시간마다 반복 수행하도록 스케줄링.

### 목적

- 매일 리마인드 하고싶은 개념이나 지식 등을 매번 알림으로 받아볼 수 있도록 하기 위해 작성.
- 인터뷰 목적의 봇보다는 개념을 까먹지 않고 익숙해질 수 있도록 하기 위해 개발함.

### 소스

- API KEY, Token 등은 env 처리하여 볼 수 없음.
- 대신 주석 달아놓았으니 참고바랍니다!
